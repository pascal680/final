BootStrap 5 est un framework très populaire en développement Web il est principalement utilisé, car il permet de faire le développement Web plus facilement et rapidement en utilisant les librairies BootStrap. Il est aussi très populaire, car il y a plusieurs façon de l'implémenter mais la façon la plus populaire est d'utilisé le CDN (Content Delivery Network) de BootStrap. Celui-ci est préférable, car il n'a pas besoin de télécharger les librairies mais plutôt de juste inclure le lien au librairie dans sa page HTML et les librairies sont automatiquement mise à jour, les librairies ne sont pas gardées localement.


Angular 11 est un framework populaire pour le développement web qui utilise une architecture de fichier prédéfinie ce qui permet de manipuler le code et les ressources facilement et efficacement. Il est un framework basé sur l'utilisation de components dans une page web et principalement utilisé pour les pages SPA (single page application). Il est aussi très populaire, car il permet l'implémentation facile avec un gestionnaire de dépendances ce qui facilitent le développement grandement. De plus, il permet de limiter le code inutile en permettant des structures efficaces et avancés de programmation.


TypeScript est le langage de programmation utilisé dans le framework Angular, de plus il peut être utilisé tout seul et compilé en tant que code javaScript. La transpilation du code TypeScript en code javaScript est ce qui a rendu la programmation TypeScript aussi populaire, car celle-ci ne donne que des avantages à la programmation web, par exemple le code typeScript permet de gérer le typages des valeurs facilement et de faciliter la programmation orientée objet grandement. C'est bénéfices de la programmation typeScript sont les raisons pour laquelle le TypeScript est si recherché et important en tant que développeur. Celui-ci amènera le futur de la programmation web c'est certain.
